# news-backtest-engine

News-driven stock backtesting engine — Finnhub API, IBKR, MT5, HMM regime filters, Monte Carlo validation, pairs trading.

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env   # then fill in your keys
```

## Environment Variables

```
  FINNHUB_API_KEY=your_finnhub_api_key_here
  STOCKNEWS_API_KEY=your_stocknews_api_key_here
  IBKR_HOST=your_ibkr_host_here
  IBKR_PORT=your_ibkr_port_here
```
